# E-Commerce Landing Page

## Overview

A responsive e-commerce landing page clone built with vanilla HTML, CSS, and JavaScript. Features include product carousels, modals, tier selection, customer reviews, and FAQ sections.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3 / Tailwind CSS** - Styling and responsive design
- **JavaScript / jQuery** - Interactivity and DOM manipulation
- **Slick Carousel** - Image sliders

## Getting Started

### Prerequisites

- <img src="https://api.iconify.design/lucide:code.svg" width="16" height="16"> [Visual Studio Code](https://code.visualstudio.com/)
- <img src="https://api.iconify.design/lucide:server.svg" width="16" height="16"> [Live Server Extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

### Installation

1. **<img src="https://api.iconify.design/lucide:git-branch.svg" width="16" height="16"> Clone the repository**

   ```bash
   git clone https://github.com/QuangNV23062004/landing-page.git
   ```

2. **<img src="https://api.iconify.design/lucide:folder-open.svg" width="16" height="16"> Open in VS Code**

   ```bash
   cd landing-page
   code .
   ```

3. **<img src="https://api.iconify.design/lucide:download.svg" width="16" height="16"> Install Live Server Extension**

   - Open VS Code
   - Go to Extensions (`Ctrl + Shift + X`)
   - Search for "Live Server"
   - Click **Install**

4. **<img src="https://api.iconify.design/lucide:play.svg" width="16" height="16"> Run the project**
   - Navigate to `pages/landing.html`
   - Right-click on the file
   - Select **"Open with Live Server"**
   - The page will open in your default browser at `http://127.0.0.1:5500`
